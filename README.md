# アリスとボブ

## はじめに
セキュリティモデルの図表に頻出する『[アリスとボブ](https://ja.wikipedia.org/wiki/%E3%82%A2%E3%83%AA%E3%82%B9%E3%81%A8%E3%83%9C%E3%83%96)』。  
ときに被害者、ときに加害者。  
そんな彼らに味付けしてみました。  
ボブくんたちに力を借りて、みなさんのイメージを楽しく伝えてみませんか？
